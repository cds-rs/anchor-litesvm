# Rendering PlantUML Diagrams

The diagrams in this book are [PlantUML](https://plantuml.com); the [Modeling with Diagrams](../modeling.md) primer covers the small slice of the language they use. To turn a `.puml` source into an image, reach for any of these:

- [PlantText](https://www.planttext.com): a browser editor with live preview, nothing to install.
- [plantuml.com](https://plantuml.com): the language reference and an online server, plus the jar and CLI to download.
- [plantuml-server on Docker Hub](https://hub.docker.com/r/plantuml/plantuml-server): run your own with `docker run -d -p 8080:8080 plantuml/plantuml-server`.
- [PlantUML integrations](https://github.com/marketplace?query=plantuml): editor and CI plugins (VS Code, JetBrains, GitHub Actions).
